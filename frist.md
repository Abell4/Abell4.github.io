### linux 基本命令
      1. pwd   查看当前位置
	  2. cd 路径  进人此文件内
	  3. cd~ 回到根目录
	  4. cd/ 回到root的根目录
	  5. ls 查看当前路径下的文件
	  6. rm -rf 有*是删除当前路径下所有文件  +文件名，删除此文件。
	  7. echo "内容" >> 文件名    把"内容"写进"文件名"里。如果没有此文件，则新建后写入。
	  8. mkdir 文件名 创建文件夹
	  9. touch 文件名 创建新文件
### git命令 (注意：创建的文件一定要.md的)
      1. git status  命令用于显示工作目录和暂存区的状态
	  2. git clone路径 克隆文件
	  3. $ ssh-keygen -t rsa -C 邮箱号"    生成ssh 密钥，输入命令后，连按三个回车即可生成ssh 密钥
	  4. git add 文件名      将文件添加到暂存区
	  5. git commit -m "标签名"   用commit推送修改到本地库中。  此标签名字可以用来当做坐标，以便回档
	  6.  git config --global user.email"邮箱号" 
	      git config --global user.name"名字" 
      7. git push -u origin master  使用-u选项指定一个默认主机，这样后面就可以不加任何参数使用Git push 	添加到服务器中
