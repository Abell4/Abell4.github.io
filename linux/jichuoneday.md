# linux命令与应用基础篇
## Linux命令的基本格式
	- 命令字 [选项] [参数]
		- 选项:用于调节命令的具体功能
		- 参数:命令操作的对象 eg:文件,目录等

## 文件与目录
	### 显示文件和目录列表(list)
		- ls  常用参数 : -l (long)  -a (all : 包括隐藏文件)  -t (time)
		
	### 帮助命令
		- 内部: help +命令
		- 外部: man +命令
